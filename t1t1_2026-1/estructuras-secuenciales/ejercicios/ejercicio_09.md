# Ejercicio 9: Área del Círculo

## Dificultad: ⭐⭐ Intermedio

## Descripción

Crea un programa que calcule el área de un círculo dado su radio.

## Fórmula

```
Área = π × radio²
```

Donde π (pi) ≈ 3.14159

## Análisis

| Parte | Descripción |
|-------|-------------|
| **ENTRADA** | Radio del círculo |
| **PROCESO** | Calcular área = pi * radio ** 2 |
| **SALIDA** | Área del círculo |

## Ejemplo de Ejecución

```
Ingresa el radio del círculo: 5
El área del círculo es: 78.53975
```

## Pistas

- Define pi como una constante: `PI = 3.14159`
- Usa `**` para elevar al cuadrado: `radio ** 2`
- También puedes usar `radio * radio`

## Reto Extra

Calcula también el perímetro (circunferencia):
```
Perímetro = 2 × π × radio
```
