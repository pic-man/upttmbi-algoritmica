# Ejercicio 3: Área del Rectángulo

## Dificultad: ⭐ Básico

## Descripción

Crea un programa que calcule el área de un rectángulo dados su base y altura.

## Fórmula

```
Área = base × altura
```

## Análisis

| Parte | Descripción |
|-------|-------------|
| **ENTRADA** | Base y altura del rectángulo |
| **PROCESO** | Calcular área = base * altura |
| **SALIDA** | Mostrar el área calculada |

## Ejemplo de Ejecución

```
Ingresa la base del rectángulo: 5
Ingresa la altura del rectángulo: 3
El área del rectángulo es: 15
```

## Pistas

- Usa `float()` para permitir valores decimales
- La fórmula es simplemente multiplicar base por altura
