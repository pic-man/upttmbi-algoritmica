# Ejercicio 5: Conversión de Temperatura

## Dificultad: ⭐ Básico

## Descripción

Crea un programa que convierta una temperatura de grados Celsius a Fahrenheit.

## Fórmula

```
°F = (°C × 9/5) + 32
```

## Análisis

| Parte | Descripción |
|-------|-------------|
| **ENTRADA** | Temperatura en grados Celsius |
| **PROCESO** | Aplicar fórmula de conversión |
| **SALIDA** | Temperatura en grados Fahrenheit |

## Ejemplo de Ejecución

```
Ingresa la temperatura en Celsius: 25
25°C equivalen a 77.0°F
```

## Tabla de Verificación

| Celsius | Fahrenheit |
|---------|------------|
| 0 | 32 |
| 100 | 212 |
| 25 | 77 |
| -40 | -40 |

## Pistas

- Usa `float()` para la entrada
- Usa paréntesis para asegurar el orden correcto de operaciones
- Puedes escribir `9/5` o `1.8` (son equivalentes)

## Entrega

Guarda tu solución como `ejercicio_05.py`
