# Ejercicio 1: Saludo Personalizado

## Dificultad: ⭐ Básico

## Descripción

Crea un programa que pida el nombre del usuario y lo salude de forma personalizada.

## Análisis

| Parte | Descripción |
|-------|-------------|
| **ENTRADA** | Nombre del usuario |
| **PROCESO** | Ninguno |
| **SALIDA** | Mensaje de saludo con el nombre |

## Ejemplo de Ejecución

```
¿Cuál es tu nombre? María
¡Hola, María! Bienvenido al curso de Algorítmica.
```

## Pistas

- Usa `input()` para pedir el nombre
- Usa `print()` con f-string para mostrar el saludo
- El nombre no necesita conversión (ya es texto)

## Entrega

Guarda tu solución como `ejercicio_01.py`
