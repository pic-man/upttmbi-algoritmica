# Ejercicio 4: Promedio de Tres Notas

## Dificultad: ⭐ Básico

## Descripción

Crea un programa que pida tres notas y calcule su promedio.

## Fórmula

```
Promedio = (nota1 + nota2 + nota3) / 3
```

## Análisis

| Parte | Descripción |
|-------|-------------|
| **ENTRADA** | Tres notas (nota1, nota2, nota3) |
| **PROCESO** | Calcular promedio = (n1 + n2 + n3) / 3 |
| **SALIDA** | Mostrar el promedio |

## Ejemplo de Ejecución

```
Ingresa la primera nota: 15
Ingresa la segunda nota: 18
Ingresa la tercera nota: 12
El promedio es: 15.0
```

## Pistas

- Usa `float()` porque las notas pueden tener decimales
- Usa paréntesis para sumar las notas antes de dividir
- El promedio generalmente tiene decimales
