# Ejercicio 7: Precio con Descuento

## Dificultad: ⭐⭐ Intermedio

## Descripción

Crea un programa que calcule el precio final de un producto aplicando un porcentaje de descuento.

## Fórmulas

```
descuento = precio × (porcentaje / 100)
precio_final = precio - descuento
```

## Análisis

| Parte | Descripción |
|-------|-------------|
| **ENTRADA** | Precio original y porcentaje de descuento |
| **PROCESO** | Calcular el descuento y restarlo al precio |
| **SALIDA** | Monto del descuento y precio final |

## Ejemplo de Ejecución

```
Precio del producto: 100
Porcentaje de descuento: 15
Descuento aplicado: 15.0
Precio final: 85.0
```

## Pistas

- Usa `float()` para ambas entradas
- Divide el porcentaje entre 100 para convertirlo a decimal
- Muestra tanto el descuento como el precio final

## Entrega

Guarda tu solución como `ejercicio_07.py`
