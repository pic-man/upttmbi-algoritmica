# Ejercicio 2: Suma de Dos Números

## Dificultad: ⭐ Básico

## Descripción

Crea un programa que pida dos números al usuario y muestre su suma.

## Análisis

| Parte | Descripción |
|-------|-------------|
| **ENTRADA** | Dos números (num1, num2) |
| **PROCESO** | Calcular la suma: resultado = num1 + num2 |
| **SALIDA** | Mostrar el resultado |

## Ejemplo de Ejecución

```
Ingresa el primer número: 15
Ingresa el segundo número: 8
La suma de 15 + 8 = 23
```

## Pistas

- Usa `int()` o `float()` para convertir la entrada a número
- Guarda el resultado en una variable antes de mostrarlo
- Muestra los números originales junto con el resultado
