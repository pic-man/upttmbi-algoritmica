# Ejercicio 11: Línea de asteriscos

## Enunciado

Haz un programa que pida un número N al usuario e imprima N asteriscos en una sola línea.

## Ejemplo de ejecución

```
¿Cuántos asteriscos? 7
*******
```

## Otro ejemplo

```
¿Cuántos asteriscos? 4
****
```

## Pista

Usa `print("*", end="")` para imprimir sin saltar de línea, o acumula en una variable.
