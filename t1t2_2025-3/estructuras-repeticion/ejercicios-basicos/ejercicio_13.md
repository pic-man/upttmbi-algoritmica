# Ejercicio 13: Pedir números hasta escribir 0

## Enunciado

Haz un programa que pida números al usuario repetidamente hasta que escriba 0. Al terminar, muestra cuántos números ingresó (sin contar el 0).

## Ejemplo de ejecución

```
Ingresa un número (0 para salir): 5
Ingresa un número (0 para salir): 8
Ingresa un número (0 para salir): 3
Ingresa un número (0 para salir): 0
Ingresaste 3 números.
```

## Pista

Usa un bucle `while` que se detenga cuando el número sea 0. Usa un contador.
