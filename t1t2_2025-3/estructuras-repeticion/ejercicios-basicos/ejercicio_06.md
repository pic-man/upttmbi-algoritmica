# Ejercicio 6: Suma del 1 al 100

## Enunciado

Haz un programa que calcule e imprima la suma de todos los números del 1 al 100.

## Salida esperada

```
La suma del 1 al 100 es: 5050
```

## Pista

Usa una variable acumuladora que empiece en 0 y vaya sumando cada número.

```python
suma = 0
for i in range(1, 101):
    suma = suma + i
```
