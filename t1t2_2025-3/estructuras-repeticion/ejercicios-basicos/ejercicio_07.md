# Ejercicio 7: Números del 1 al N

## Enunciado

Haz un programa que pida un número N al usuario e imprima todos los números del 1 hasta N.

## Ejemplo de ejecución

```
Ingresa un número: 5
1
2
3
4
5
```

## Otro ejemplo

```
Ingresa un número: 3
1
2
3
```

## Pista

Usa `int(input())` para pedir el número y luego `range(1, n+1)`.
