# Ejercicio 14: Sumar números hasta escribir -1

## Enunciado

Haz un programa que pida números al usuario y los vaya sumando. El programa termina cuando el usuario escribe -1. Muestra la suma total (sin incluir el -1).

## Ejemplo de ejecución

```
Ingresa un número (-1 para terminar): 10
Ingresa un número (-1 para terminar): 5
Ingresa un número (-1 para terminar): 3
Ingresa un número (-1 para terminar): -1
La suma total es: 18
```

## Pista

Usa un acumulador `suma = 0` y un `while` que termine cuando el número sea -1.
