# Ejercicios: Cadenas de Caracteres

Colección de retos prácticos organizados por dificultad. Sugerencias:

- Prefijo numérico (`01_`, `02_`, …) para ordenar los ejercicios.
- Incluir en cada archivo objetivos, instrucciones y ejemplos de entrada/salida.
- Usar `plantilla_solucion.py` como base para entregar soluciones guiadas al alumnado.
